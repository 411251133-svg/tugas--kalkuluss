Kalkulator Kalkulus

Hitung berbagai rumus kalkulus dengan mudah

Kalkulator Pertidaksamaan Linear

Selesaikan: ax + b < c

Penyelesaian:

3x + 5 < 17

3x < 12

x < 4

Himpunan Penyelesaian: x < 4

Kalkulator Gradien

Gradien Melalui Dua Titik

Hasil:

m = (y₂ - y₁) / (x₂ - x₁)

m = (9 - 3) / (5 - 2)

m = 6 / 3

Gradien m = 2

Kalkulator Fungsi Kuadrat

Analisis y = ax² + bx + c

Hasil Analisis:

Fungsi: y = x² - 4x + 3

Titik Puncak x = -b/(2a) = 4/2 = 2

Titik Puncak y = 1(2²) - 4(2) + 3 = -1

Puncak: (2, -1)

Sumbu Simetri: x = 2

Titik Potong Sumbu Y: (0, 3)

Kalkulator Fungsi Rasional

Analisis f(x) = (ax + b)/(cx + d)

Hasil:

f(x) = (2x + 1) / (x - 3)

Domain: x ≠ 3

Asimtot Vertikal: x = 3

Asimtot Horizontal: y = a/c = 2/1 = 2

AV: x = 3, AH: y = 2

Kalkulator Komposisi Fungsi

(f ∘ g)(x) untuk f(x) = ax + b dan g(x) = cx + d

Hasil:

f(x) = 2x + 1

g(x) = 3x - 2

(f ∘ g)(x) = f(g(x))

= f(3x - 2)

= 2(3x - 2) + 1

= 6x - 4 + 1

(f ∘ g)(x) = 6x - 3

Kalkulator Kekontinuan

Cek Kekontinuan f(x) = ax² + bx + c di x = k

Analisis:

f(x) = x² + 2x + 3

1. f(2) = 1(4) + 2(2) + 3 = 11 ✓

2. lim(x→2) f(x) = 11 ✓

3. lim(x→2) f(x) = f(2) ✓

Fungsi KONTINU di x = 2

Kalkulator Limit

lim(x→a) (ax² + bx + c)

Hasil:

lim(x→3) (x² + 2x - 1)

Substitusi langsung:

= (3)² + 2(3) - 1

= 9 + 6 - 1

Limit = 14

Kalkulator Turunan

Turunan f(x) = ax³ + bx² + cx + d

Hasil:

f(x) = 3x³ - 2x² + 5x - 1

f'(x) = 3(3x²) - 2(2x) + 5(1) - 0

f'(x) = 9x² - 4x + 5

Kalkulator Titik Ekstrim

Titik Ekstrim f(x) = ax² + bx + c

Hasil:

f(x) = x² - 4x + 3

f'(x) = 2x - 4 = 0

x = 2

f(2) = (2)² - 4(2) + 3 = -1

f''(x) = 2 > 0 (minimum)

Minimum di (2, -1)

Kalkulator Integral

∫ (ax² + bx + c) dx

Hasil:

∫ (3x² + 2x - 1) dx

= 3(x³/3) + 2(x²/2) - x + C

= x³ + x² - x + C

Integral Tentu ∫ₐᵇ (px² + qx + r) dx

Hasil:

∫₀² (2x² + x) dx

= [2x³/3 + x²/2]₀²

= (16/3 + 2) - (0)

= 16/3 + 6/3 = 22/3

≈ 7.33 satuan luas
