Soal Latihan Pertidaksamaan

Soal 1

Tentukan himpunan penyelesaian dari 2x + 7 < 15

Lihat Pembahasan

2x + 7 < 15

2x < 15 - 7

2x < 8

x < 4

Jawaban: x < 4 atau (-∞, 4)

Soal 2

Selesaikan pertidaksamaan: -3x + 5 ≥ 14

Lihat Pembahasan

-3x + 5 ≥ 14

-3x ≥ 14 - 5

-3x ≥ 9

x ≤ -3 (tanda berubah karena dibagi negatif)

Jawaban: x ≤ -3 atau (-∞, -3]

Soal 3

Tentukan nilai x yang memenuhi: 5 ≤ 2x - 1 ≤ 11

Lihat Pembahasan

5 ≤ 2x - 1 ≤ 11

5 + 1 ≤ 2x ≤ 11 + 1

6 ≤ 2x ≤ 12

3 ≤ x ≤ 6

Jawaban: 3 ≤ x ≤ 6 atau [3, 6]

Soal 4

Selesaikan: x² - 5x + 6 > 0

Lihat Pembahasan

x² - 5x + 6 > 0

(x - 2)(x - 3) > 0

Titik kritis: x = 2 dan x = 3

Uji interval: x < 2 (+), 2 < x < 3 (-), x > 3 (+)

Jawaban: x < 2 atau x > 3

Soal 5

Tentukan HP dari |x - 3| < 5

Lihat Pembahasan

|x - 3| < 5

-5 < x - 3 < 5

-5 + 3 < x < 5 + 3

-2 < x < 8

Jawaban: -2 < x < 8 atau (-2, 8)

Soal Latihan Gradien

Soal 1

Tentukan gradien garis yang melalui titik A(1, 3) dan B(4, 9)

Lihat Pembahasan

m = (y₂ - y₁) / (x₂ - x₁)

m = (9 - 3) / (4 - 1)

m = 6 / 3

Jawaban: m = 2

Soal 2

Tentukan gradien garis dengan persamaan 3x + 4y = 12

Lihat Pembahasan

3x + 4y = 12

4y = -3x + 12

y = -3/4 x + 3

Jawaban: m = -3/4

Soal 3

Tentukan persamaan garis yang melalui (2, 5) dengan gradien 3

Lihat Pembahasan

y - y₁ = m(x - x₁)

y - 5 = 3(x - 2)

y - 5 = 3x - 6

y = 3x - 1

Jawaban: y = 3x - 1

Soal 4

Tentukan gradien garis yang tegak lurus dengan garis y = 2x + 5

Lihat Pembahasan

Gradien garis y = 2x + 5 adalah m₁ = 2

Untuk garis tegak lurus: m₁ × m₂ = -1

2 × m₂ = -1

m₂ = -1/2

Jawaban: m = -1/2

Soal 5

Tentukan persamaan garis yang sejajar y = -4x + 7 dan melalui (3, 1)

Lihat Pembahasan

Garis sejajar memiliki gradien sama: m = -4

y - y₁ = m(x - x₁)

y - 1 = -4(x - 3)

y - 1 = -4x + 12

y = -4x + 13

Jawaban: y = -4x + 13

Soal Latihan Grafik Fungsi

Soal 1

Tentukan titik potong grafik y = 2x - 4 dengan sumbu x dan y

Lihat Pembahasan

Titik potong sumbu y (x = 0):

y = 2(0) - 4 = -4 → (0, -4)

Titik potong sumbu x (y = 0):

0 = 2x - 4

2x = 4

x = 2 → (2, 0)

Jawaban: Sumbu y: (0, -4), Sumbu x: (2, 0)

Soal 2

Tentukan koordinat puncak parabola y = x² - 4x + 3

Lihat Pembahasan

Untuk y = ax² + bx + c

x puncak = -b/(2a) = -(-4)/(2×1) = 2

y puncak = (2)² - 4(2) + 3 = 4 - 8 + 3 = -1

Jawaban: Puncak di (2, -1)

Soal 3

Tentukan persamaan sumbu simetri dari y = -x² + 6x - 5

Lihat Pembahasan

Sumbu simetri: x = -b/(2a)

x = -6/(2×(-1))

x = -6/(-2)

x = 3

Jawaban: x = 3

Soal 4

Grafik y = ax² + bx + c melalui (0, 3), (1, 0), dan (3, 0). Tentukan nilai a, b, dan c

Lihat Pembahasan

Dari (0, 3): c = 3

Akar-akar: x = 1 dan x = 3

y = a(x - 1)(x - 3)

3 = a(0 - 1)(0 - 3) = 3a

a = 1

y = (x - 1)(x - 3) = x² - 4x + 3

Jawaban: a = 1, b = -4, c = 3

Soal 5

Tentukan nilai maksimum dari fungsi y = -2x² + 8x - 3

Lihat Pembahasan

x maksimum = -b/(2a) = -8/(2×(-2)) = 2

y maksimum = -2(2)² + 8(2) - 3

= -8 + 16 - 3

= 5

Jawaban: Nilai maksimum = 5 di x = 2

Soal Latihan Fungsi Rasional

Soal 1

Tentukan domain dari f(x) = 1/(x - 5)

Lihat Pembahasan

Domain: semua x kecuali penyebut = 0

x - 5 ≠ 0

x ≠ 5

Jawaban: Domain = {x | x ≠ 5} atau ℝ - {5}

Soal 2

Tentukan asimtot vertikal dan horizontal dari f(x) = (2x + 1)/(x - 3)

Lihat Pembahasan

Asimtot Vertikal: x - 3 = 0 → x = 3

Asimtot Horizontal:

Derajat pembilang = derajat penyebut

y = 2/1 = 2

Jawaban: AV: x = 3, AH: y = 2

Soal 3

Sederhanakan (x² - 9)/(x - 3)

Lihat Pembahasan

(x² - 9)/(x - 3)

= [(x + 3)(x - 3)]/(x - 3)

= x + 3, dengan x ≠ 3

Jawaban: x + 3, x ≠ 3

Soal 4

Tentukan range dari f(x) = 3/(x + 2)

Lihat Pembahasan

y = 3/(x + 2)

y(x + 2) = 3

xy + 2y = 3

xy = 3 - 2y

x = (3 - 2y)/y (y ≠ 0)

Jawaban: Range = {y | y ≠ 0} atau ℝ - {0}

Soal 5

Tentukan nilai f(2) jika f(x) = (x² + x - 6)/(x² - 4)

Lihat Pembahasan

f(2) = (2² + 2 - 6)/(2² - 4)

= (4 + 2 - 6)/(4 - 4)

= 0/0 (tak tentu)

Faktorkan: [(x + 3)(x - 2)]/[(x + 2)(x - 2)]

= (x + 3)/(x + 2)

f(2) = (2 + 3)/(2 + 2) = 5/4

Jawaban: f(2) = 5/4

Soal Latihan Komposisi Fungsi

Soal 1

Jika f(x) = 2x + 3 dan g(x) = x - 1, tentukan (f ∘ g)(x)

Lihat Pembahasan

(f ∘ g)(x) = f(g(x))

= f(x - 1)

= 2(x - 1) + 3

= 2x - 2 + 3

= 2x + 1

Jawaban: (f ∘ g)(x) = 2x + 1

Soal 2

Jika f(x) = x² dan g(x) = 3x + 2, tentukan (g ∘ f)(2)

Lihat Pembahasan

(g ∘ f)(x) = g(f(x)) = g(x²)

= 3(x²) + 2

= 3x² + 2

(g ∘ f)(2) = 3(2)² + 2

= 3(4) + 2 = 14

Jawaban: (g ∘ f)(2) = 14

Soal 3

Tentukan f⁻¹(x) jika f(x) = (2x - 1)/3

Lihat Pembahasan

y = (2x - 1)/3

3y = 2x - 1

2x = 3y + 1

x = (3y + 1)/2

f⁻¹(x) = (3x + 1)/2

Jawaban: f⁻¹(x) = (3x + 1)/2

Soal 4

Jika (f ∘ g)(x) = 4x² + 6x + 2 dan g(x) = 2x + 1, tentukan f(x)

Lihat Pembahasan

Misalkan g(x) = u, maka 2x + 1 = u

x = (u - 1)/2

f(u) = 4[(u - 1)/2]² + 6[(u - 1)/2] + 2

= 4(u² - 2u + 1)/4 + 3(u - 1) + 2

= u² - 2u + 1 + 3u - 3 + 2

= u² + u

Jawaban: f(x) = x² + x

Soal 5

Jika f(x) = x + 5 dan (f ∘ g)(x) = 3x² - 2, tentukan g(x)

Lihat Pembahasan

(f ∘ g)(x) = f(g(x)) = g(x) + 5 = 3x² - 2

g(x) = 3x² - 2 - 5

g(x) = 3x² - 7

Jawaban: g(x) = 3x² - 7

Soal Latihan Kekontinuan Fungsi

Soal 1

Periksa kekontinuan f(x) = x² + 3x - 1 di x = 2

Lihat Pembahasan

1. f(2) = 2² + 3(2) - 1 = 4 + 6 - 1 = 9 (terdefinisi)

2. lim (x→2) f(x) = 2² + 3(2) - 1 = 9 (ada)

3. lim (x→2) f(x) = f(2) = 9 (sama)

Jawaban: Fungsi kontinu di x = 2

Soal 2

Tentukan apakah f(x) = 1/(x - 3) kontinu di x = 3

Lihat Pembahasan

f(3) = 1/(3 - 3) = 1/0 (tidak terdefinisi)

Karena f(3) tidak terdefinisi, fungsi tidak kontinu di x = 3

Jawaban: Tidak kontinu (diskontinuitas tak hingga)

Soal 3

Tentukan nilai a agar f(x) kontinu di x = 2 jika f(x) = {x² + 1, x < 2; ax + 3, x ≥ 2}

Lihat Pembahasan

Agar kontinu: lim (x→2⁻) f(x) = lim (x→2⁺) f(x) = f(2)

lim (x→2⁻) = 2² + 1 = 5

lim (x→2⁺) = a(2) + 3 = 2a + 3

5 = 2a + 3

2a = 2

a = 1

Jawaban: a = 1

Soal 4

Apakah f(x) = |x| kontinu di x = 0?

Lihat Pembahasan

f(0) = |0| = 0 (terdefinisi)

lim (x→0⁻) |x| = 0

lim (x→0⁺) |x| = 0

lim (x→0) |x| = 0 = f(0)

Jawaban: Ya, kontinu di x = 0

Soal 5

Tentukan interval kekontinuan f(x) = √(x - 4)

Lihat Pembahasan

Fungsi akar kuadrat terdefinisi jika x - 4 ≥ 0

x ≥ 4

Fungsi kontinu pada domainnya

Jawaban: Kontinu pada [4, ∞)

Soal Latihan Limit

Soal 1

Tentukan lim (x→3) (x² + 2x - 1)

Lihat Pembahasan

Substitusi langsung:

lim (x→3) (x² + 2x - 1)

= 3² + 2(3) - 1

= 9 + 6 - 1 = 14

Jawaban: 14

Soal 2

Tentukan lim (x→2) (x² - 4)/(x - 2)

Lihat Pembahasan

Bentuk 0/0, faktorkan:

= lim (x→2) [(x + 2)(x - 2)]/(x - 2)

= lim (x→2) (x + 2)

= 2 + 2 = 4

Jawaban: 4

Soal 3

Tentukan lim (x→∞) (3x² + 2x - 1)/(x² + 5)

Lihat Pembahasan

Bagi dengan pangkat tertinggi (x²):

= lim (x→∞) (3 + 2/x - 1/x²)/(1 + 5/x²)

= (3 + 0 - 0)/(1 + 0)

= 3/1 = 3

Jawaban: 3

Soal 4

Tentukan lim (x→0) (√(x + 4) - 2)/x

Lihat Pembahasan

Kalikan dengan sekawan:

= lim (x→0) [(√(x + 4) - 2)(√(x + 4) + 2)]/[x(√(x + 4) + 2)]

= lim (x→0) [(x + 4) - 4]/[x(√(x + 4) + 2)]

= lim (x→0) x/[x(√(x + 4) + 2)]

= lim (x→0) 1/(√(x + 4) + 2)

= 1/(√4 + 2) = 1/4

Jawaban: 1/4

Soal 5

Tentukan lim (x→0) sin(3x)/x

Lihat Pembahasan

Gunakan lim (x→0) sin(ax)/x = a:

lim (x→0) sin(3x)/x

= lim (x→0) 3 × sin(3x)/(3x)

= 3 × 1 = 3

Jawaban: 3

Soal Latihan Turunan

Soal 1

Tentukan turunan f(x) = 5x³ - 3x² + 7x - 2

Lihat Pembahasan

f'(x) = d/dx (5x³ - 3x² + 7x - 2)

= 5(3x²) - 3(2x) + 7(1) - 0

= 15x² - 6x + 7

Jawaban: f'(x) = 15x² - 6x + 7

Soal 2

Tentukan turunan f(x) = (2x + 1)³

Lihat Pembahasan

Gunakan aturan rantai:

f'(x) = 3(2x + 1)² × 2

= 6(2x + 1)²

Jawaban: f'(x) = 6(2x + 1)²

Soal 3

Tentukan turunan f(x) = x²(3x - 1)

Lihat Pembahasan

Gunakan aturan perkalian: (uv)' = u'v + uv'

u = x², u' = 2x

v = 3x - 1, v' = 3

f'(x) = 2x(3x - 1) + x²(3)

= 6x² - 2x + 3x²

= 9x² - 2x

Jawaban: f'(x) = 9x² - 2x

Soal 4

Tentukan turunan f(x) = (x + 1)/(2x - 3)

Lihat Pembahasan

Gunakan aturan pembagian: (u/v)' = (u'v - uv')/v²

u = x + 1, u' = 1

v = 2x - 3, v' = 2

f'(x) = [1(2x - 3) - (x + 1)(2)]/(2x - 3)²

= (2x - 3 - 2x - 2)/(2x - 3)²

= -5/(2x - 3)²

Jawaban: f'(x) = -5/(2x - 3)²

Soal 5

Jika f(x) = x³ - 2x, tentukan f'(2)

Lihat Pembahasan

f'(x) = 3x² - 2

f'(2) = 3(2)² - 2

= 3(4) - 2

= 12 - 2 = 10

Jawaban: f'(2) = 10

Soal Latihan Titik Ekstrim

Soal 1

Tentukan titik stasioner dari f(x) = x² - 4x + 3

Lihat Pembahasan

f'(x) = 2x - 4 = 0

2x = 4

x = 2

f(2) = 2² - 4(2) + 3 = 4 - 8 + 3 = -1

Jawaban: Titik stasioner di (2, -1)

Soal 2

Tentukan jenis ekstrim dari f(x) = -x² + 6x - 5

Lihat Pembahasan

f'(x) = -2x + 6 = 0 → x = 3

f''(x) = -2 < 0 (maksimum)

f(3) = -(3)² + 6(3) - 5 = -9 + 18 - 5 = 4

Jawaban: Maksimum di (3, 4)

Soal 3

Tentukan titik ekstrim f(x) = x³ - 6x² + 9x + 1

Lihat Pembahasan

f'(x) = 3x² - 12x + 9 = 0

3(x² - 4x + 3) = 0

3(x - 1)(x - 3) = 0

x = 1 atau x = 3

f''(x) = 6x - 12

f''(1) = -6 < 0 → maks di (1, 5)

f''(3) = 6 > 0 → min di (3, 1)

Jawaban: Maks (1, 5), Min (3, 1)

Soal 4

Tentukan nilai maksimum fungsi f(x) = -2x² + 8x - 3 pada interval [0, 5]

Lihat Pembahasan

f'(x) = -4x + 8 = 0 → x = 2

f(0) = -3

f(2) = -2(4) + 8(2) - 3 = 5

f(5) = -2(25) + 8(5) - 3 = -13

Jawaban: Nilai maksimum = 5 di x = 2

Soal 5

Sebuah persegi panjang memiliki keliling 40 cm. Tentukan dimensi yang memberikan luas maksimum

Lihat Pembahasan

Keliling: 2p + 2l = 40 → p + l = 20

l = 20 - p

Luas: A = p × l = p(20 - p) = 20p - p²

A' = 20 - 2p = 0 → p = 10

l = 20 - 10 = 10

Jawaban: Persegi 10 cm × 10 cm

Soal Latihan Integral

Soal 1

Tentukan ∫ (4x³ - 6x² + 2x - 1) dx

Lihat Pembahasan

∫ (4x³ - 6x² + 2x - 1) dx

= 4(x⁴/4) - 6(x³/3) + 2(x²/2) - x + C

= x⁴ - 2x³ + x² - x + C

Jawaban: x⁴ - 2x³ + x² - x + C

Soal 2

Tentukan ∫₀² (3x² + 2x) dx

Lihat Pembahasan

∫ (3x² + 2x) dx = x³ + x²

∫₀² (3x² + 2x) dx = [x³ + x²]₀²

= (2³ + 2²) - (0³ + 0²)

= (8 + 4) - 0 = 12

Jawaban: 12

Soal 3

Tentukan ∫ (2x + 1)³ dx

Lihat Pembahasan

Misalkan u = 2x + 1, maka du = 2dx

∫ (2x + 1)³ dx = ∫ u³ (du/2)

= (1/2) × (u⁴/4) + C

= (2x + 1)⁴/8 + C

Jawaban: (2x + 1)⁴/8 + C

Soal 4

Hitunglah luas daerah yang dibatasi y = x², sumbu x, x = 0, dan x = 3

Lihat Pembahasan

Luas = ∫₀³ x² dx

= [x³/3]₀³

= 3³/3 - 0³/3

= 27/3 = 9

Jawaban: 9 satuan luas

Soal 5

Jika f'(x) = 6x² - 4x + 1 dan f(0) = 5, tentukan f(x)

Lihat Pembahasan

f(x) = ∫ (6x² - 4x + 1) dx

= 2x³ - 2x² + x + C

f(0) = 0 - 0 + 0 + C = 5

C = 5

f(x) = 2x³ - 2x² + x + 5

Jawaban: f(x) = 2x³ - 2x² + x + 5
