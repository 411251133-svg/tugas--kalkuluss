Materi 01

Pertidaksamaan

Pengertian

Pertidaksamaan adalah kalimat terbuka yang menggunakan tanda ketidaksamaan seperti <, >, ≤, atau ≥.

Sifat-sifat Pertidaksamaan

  • Jika a < b maka a + c < b + c
  • Jika a < b dan c > 0 maka ac < bc
  • Jika a < b dan c < 0 maka ac > bc
  • Jika a < b < 0 maka 1/a > 1/b

Contoh Soal

Selesaikan: 3x + 5 < 17

Penyelesaian:

3x + 5 < 17

3x < 17 - 5

3x < 12

x < 4

Jadi, himpunan penyelesaiannya adalah x < 4

Materi 02

Gradien

Pengertian

Gradien adalah nilai yang menyatakan tingkat kemiringan suatu garis terhadap sumbu x positif.

Rumus Gradien

Gradien garis melalui dua titik (x₁, y₁) dan (x₂, y₂):

m = (y₂ - y₁) / (x₂ - x₁)

Gradien garis y = mx + c:

m adalah gradien

Sifat-sifat Gradien

  • Dua garis sejajar memiliki gradien yang sama
  • Dua garis tegak lurus, perkalian gradiennya = -1
  • Jika m > 0, garis naik dari kiri ke kanan
  • Jika m < 0, garis turun dari kiri ke kanan

Contoh Soal

Tentukan gradien garis melalui (2, 3) dan (5, 9)

Penyelesaian:

m = (9 - 3) / (5 - 2)

m = 6 / 3

m = 2

Jadi, gradiennya adalah 2

Materi 03

Grafik Fungsi

Pengertian

Grafik fungsi adalah himpunan titik-titik pada bidang koordinat yang merepresentasikan hubungan antara variabel bebas (x) dan variabel terikat (y).

Jenis-jenis Grafik Fungsi

  • Fungsi Linear: y = mx + c (garis lurus)
  • Fungsi Kuadrat: y = ax² + bx + c (parabola)
  • Fungsi Kubik: y = ax³ + bx² + cx + d
  • Fungsi Eksponen: y = aˣ
  • Fungsi Logaritma: y = log(x)

Langkah Menggambar Grafik

  1. Tentukan titik potong dengan sumbu x dan y
  2. Tentukan titik-titik penting lainnya
  3. Hubungkan titik-titik tersebut
  4. Perhatikan sifat khusus grafik

Contoh

Grafik y = x² - 4x + 3

Titik potong sumbu y: (0, 3)

Titik potong sumbu x: (1, 0) dan (3, 0)

Titik puncak: (2, -1)

Materi 04

Fungsi Rasional

Pengertian

Fungsi rasional adalah fungsi yang berbentuk pecahan dengan pembilang dan penyebut merupakan fungsi polinom.

f(x) = P(x) / Q(x)

dimana P(x) dan Q(x) adalah polinom, Q(x) ≠ 0

Domain Fungsi Rasional

Domain adalah semua nilai x yang membuat penyebut tidak sama dengan nol.

Asimtot

  • Asimtot Vertikal: x = a, dimana Q(a) = 0
  • Asimtot Horizontal: ditentukan oleh derajat P(x) dan Q(x)
  • Asimtot Miring: jika derajat P(x) = derajat Q(x) + 1

Contoh

f(x) = (x + 2) / (x - 3)

Domain: x ≠ 3

Asimtot vertikal: x = 3

Asimtot horizontal: y = 1

Materi 05

Komposisi Fungsi

Pengertian

Komposisi fungsi adalah penggabungan dua fungsi atau lebih menjadi satu fungsi baru.

Notasi: (f ∘ g)(x) = f(g(x))

dibaca: "f bundaran g dari x"

Sifat Komposisi Fungsi

  • Tidak komutatif: f ∘ g ≠ g ∘ f
  • Asosiatif: (f ∘ g) ∘ h = f ∘ (g ∘ h)
  • Memiliki identitas: f ∘ I = I ∘ f = f

Fungsi Invers

Jika f ∘ f⁻¹ = f⁻¹ ∘ f = I, maka f⁻¹ adalah invers dari f

Contoh

Jika f(x) = 2x + 1 dan g(x) = x², tentukan (f ∘ g)(x)

Penyelesaian:

(f ∘ g)(x) = f(g(x))

= f(x²)

= 2(x²) + 1

= 2x² + 1

Materi 06

Kekontinuan Fungsi

Pengertian

Fungsi f(x) kontinu di x = a jika limit kiri = limit kanan = nilai fungsi di titik tersebut.

Syarat Fungsi Kontinu di x = a:

  1. f(a) terdefinisi
  2. lim (x→a) f(x) ada
  3. lim (x→a) f(x) = f(a)

Jenis Diskontinuitas

  • Diskontinuitas Terlepas: ada gap/celah
  • Diskontinuitas Lompat: limit kiri ≠ limit kanan
  • Diskontinuitas Tak Hingga: fungsi mendekati ∞

Teorema Nilai Antara

Jika f kontinu pada [a,b] dan f(a) ≠ f(b), maka untuk setiap nilai k antara f(a) dan f(b), terdapat c ∈ (a,b) sehingga f(c) = k.

Contoh

Periksa kekontinuan f(x) = x² di x = 2

1. f(2) = 4 (terdefinisi)

2. lim (x→2) x² = 4 (ada)

3. lim (x→2) x² = f(2) = 4

Jadi, fungsi kontinu di x = 2

Materi 07

Limit

Pengertian

Limit adalah nilai yang didekati oleh fungsi saat variabel mendekati suatu nilai tertentu.

Notasi: lim (x→a) f(x) = L

dibaca: "limit f(x) untuk x mendekati a sama dengan L"

Sifat-sifat Limit

  • lim (x→a) [f(x) + g(x)] = lim f(x) + lim g(x)
  • lim (x→a) [f(x) × g(x)] = lim f(x) × lim g(x)
  • lim (x→a) [c × f(x)] = c × lim f(x)
  • lim (x→a) [f(x) / g(x)] = lim f(x) / lim g(x), jika lim g(x) ≠ 0

Bentuk Tak Tentu

  • 0/0 - gunakan faktorisasi atau L'Hospital
  • ∞/∞ - bagi dengan pangkat tertinggi
  • ∞ - ∞ - rasionalkan atau ubah bentuk

Contoh

Tentukan lim (x→2) (x² - 4) / (x - 2)

Penyelesaian:

= lim (x→2) [(x + 2)(x - 2)] / (x - 2)

= lim (x→2) (x + 2)

= 2 + 2 = 4

Materi 08

Turunan

Pengertian

Turunan adalah laju perubahan sesaat dari suatu fungsi terhadap variabelnya.

Definisi Turunan:

f'(x) = lim (h→0) [f(x+h) - f(x)] / h

Aturan Dasar Turunan

  • f(x) = c → f'(x) = 0
  • f(x) = xⁿ → f'(x) = n × xⁿ⁻¹
  • f(x) = c × g(x) → f'(x) = c × g'(x)
  • f(x) = u(x) + v(x) → f'(x) = u'(x) + v'(x)

Aturan Rantai

Jika y = f(u) dan u = g(x), maka:

dy/dx = dy/du × du/dx

Aturan Perkalian dan Pembagian

  • Perkalian: (u × v)' = u' × v + u × v'
  • Pembagian: (u/v)' = (u' × v - u × v') / v²

Contoh

Tentukan turunan f(x) = 3x⁴ - 2x² + 5

Penyelesaian:

f'(x) = 3 × 4x³ - 2 × 2x + 0

f'(x) = 12x³ - 4x

Materi 09

Titik Ekstrim

Pengertian

Titik ekstrim adalah titik dimana fungsi mencapai nilai maksimum atau minimum lokal.

Syarat Titik Ekstrim

Syarat Perlu: f'(x) = 0 atau tidak terdefinisi

Syarat Cukup: menggunakan uji turunan kedua

Uji Turunan Pertama

  • Jika f' berubah dari + ke -, maka x adalah maksimum lokal
  • Jika f' berubah dari - ke +, maka x adalah minimum lokal
  • Jika f' tidak berubah tanda, maka x bukan ekstrim

Uji Turunan Kedua

  • Jika f'(a) = 0 dan f''(a) < 0, maka x = a adalah maksimum
  • Jika f'(a) = 0 dan f''(a) > 0, maka x = a adalah minimum
  • Jika f''(a) = 0, uji tidak konklusif

Contoh

Tentukan titik ekstrim f(x) = x³ - 3x² + 4

Penyelesaian:

f'(x) = 3x² - 6x = 0

3x(x - 2) = 0

x = 0 atau x = 2

f''(x) = 6x - 6

f''(0) = -6 < 0 → maksimum di (0, 4)

f''(2) = 6 > 0 → minimum di (2, 0)

Materi 10

Integral

Pengertian

Integral adalah kebalikan dari turunan (anti-turunan) atau proses mencari luas daerah di bawah kurva.

Jenis Integral

  • Integral Tak Tentu: ∫ f(x) dx = F(x) + C
  • Integral Tentu: ∫ᵇₐ f(x) dx = F(b) - F(a)

Rumus Dasar Integral

  • ∫ k dx = kx + C
  • ∫ xⁿ dx = xⁿ⁺¹/(n+1) + C, n ≠ -1
  • ∫ 1/x dx = ln|x| + C
  • ∫ eˣ dx = eˣ + C
  • ∫ sin x dx = -cos x + C
  • ∫ cos x dx = sin x + C

Teknik Integral

  • Substitusi: untuk integral komposisi fungsi
  • Parsial: ∫ u dv = uv - ∫ v du
  • Pecahan Parsial: untuk fungsi rasional

Contoh

Tentukan ∫ (3x² + 2x - 1) dx

Penyelesaian:

= 3 × x³/3 + 2 × x²/2 - x + C

= x³ + x² - x + C
